# Long Time No See!

HTML 유사 프로그래밍 언어, Tteok과 리스프 방언인 Hy에서 영향을 받은 언어입니다.

```
<import>math</import>

<set>g <math.gcd>4 6</math.gcd></set>
<if>
  <equal>g 2</equal>
  <print sep=","><.split>"Wow Fantastic!"</.split></print> <!-- Wow, Fantastic! -->
  <print sep=",">"Wow" "It's still fantastic!"</print>
</if>
```
